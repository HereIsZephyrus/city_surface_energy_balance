## 快速导航

### 我是新用户，应该从哪里开始？

1. **快速了解模型**：阅读 [算法思想.md](./算法思想.md)
2. **准备数据**：查看 [使用指南.md](./使用指南.md) 的数据准备部分
3. **运行示例**：按照 [使用指南.md](./使用指南.md) 的快速开始部分

### 我需要理解数学原理

阅读 [数学模型.md](./数学模型.md)，包含所有数学公式和物理方程。

### 我需要修改代码

1. 阅读 [具体实现.md](./具体实现.md) 了解代码结构
2. 查看相关模块的代码注释
3. 参考 [算法思想.md](./算法思想.md) 理解设计思路

### 我遇到了问题

1. 查看 [使用指南.md](./使用指南.md) 的常见问题部分
2. 检查代码注释和文档字符串
3. 参考 [具体实现.md](./具体实现.md) 的调试工具部分
